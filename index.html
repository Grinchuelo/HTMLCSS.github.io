HTML

HyperText Markup Language

What is it?

HTML is a typing language used for web development. It helps us to make the estructure of the website adding the text and the multimedia files inside of boxes strategically positionated to add styles on them later.

Origin

In the very beginning, Tim Berners-Lee, a physicist, created this "hipertext" system with the idea of sharing simple text with other scientists. So, the time passed and now we're at the version number 5 of HTML, where we can do other complex things like controlling tables, defining IDs for every element in the page and being able to linking it with another programming languages, getting the most out of it.
